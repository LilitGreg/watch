== Description ==

<br><br><iframe width="560" height="315" src="https://www.youtube.com/embed/2Wq69qrVWVs" frameborder="0" allowfullscreen></iframe><br><br>

<p>This Plugin Adds Deals to the WooCommerce Store. All you need is to select products from the admin panel and offer deals on it. This plugin will create a special page on the Website and display all deals with count down timer</p>

<ul>

<li>Add Deals of the WooCommerce Shop</li>
<li>Add Products from Multiple Categories</li>
<li>Select End time of the Deal</li>
<li>Count down timer</li>
<li>Select products and Rest will Plugin manage</li>


</ul>
